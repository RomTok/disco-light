Features

Currently the following features are supported:
  • Graphics support for DirectFB, X11 and different framebuffer implementations (Vesa, DaVinci, OMAP)
  • Input-Handling for Touchscreens and Remote-Controls
  • Event-based communication
  • flexible GUI development
  • XML-based dialog descriptions
  • Themeable and skinnable
  • Streaming of media content through xine library or gstreamer
  • SQLite3, MySQL and ODBC database support
  • basic Flash support
  • Filetransfer
