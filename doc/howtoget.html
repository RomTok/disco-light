How to get it

You can download the latest release at http://www.diskohq.org/download or if you want to stay bleeding edge, fetch the latest sources from our git repository.

To access the git repository, you have to install the git client, i.e. by issuing sudo aptitude install git-core on Debian-based systems. If your preferred distribution doesn't provide git, then download it at http://git-scm.com.

If you have installed git, you can get our latest sources by entering: 
# git clone git://www.diskohq.org/disko.git
This will create a directory called disko which contains the sources. To update them, change into the directory and call git pull.
